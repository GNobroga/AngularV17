Host elements - Pra ligar propriedades e eventos

A propriedade host contida dentro do componente refere-se ao próprio componente e serve pra aplicar atributos para ele, seja atributos de acessibilidade ou até mesmo eventos. É basicamente uma forma nova de ser usar o HostBinding e HostListener

Ciclo de vida

OnChanges
Sempre que um valor externo (através de input) entra para o componente a função ngOnChanges é acionada.
OnInit
ngOnInit é acionado quando o componente é criado, como se fosse o constructor
DoCheck
ngDoCheck é acionado sempre que um valor é modificado dentro do componente (Ex: valor de uma tag input)
AfterViewInit
É chamado após a inicialiazação da View, ou seja, depois de renderizar os elementos.
AfterContentInit
É chamado após a inicialiazação dos elementos de dentro do ng-content
AfterContentChecked
Ele funciona como o DoCheck, mas aguardando todos as entradas serem processadas
OnDestroy
Executa sempre quando o componente é destruido
ChangeDetectionStrategy.OnPush
Permite melhorar a perfomance do Angular para entrade Input nos componentes.
